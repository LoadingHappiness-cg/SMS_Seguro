Política de Privacidade — SMS Seguro

Deteção offline Sem tracking Open source

O SMS Seguro é uma aplicação Android open source criada para ajudar a detetar mensagens SMS potencialmente fraudulentas (smishing), com foco em padrões comuns em Portugal (ex.: links suspeitos, pedidos de pagamento Multibanco, pedidos de códigos).

1) Recolha de dados

O SMS Seguro não recolhe dados pessoais e não cria contas de utilizador.

  • Não envia o conteúdo das mensagens para servidores.
  • Não utiliza analytics, publicidade ou tracking.
  • Não partilha dados com terceiros.

2) Acesso às Notificações

O SMS Seguro pode pedir Acesso às Notificações para conseguir analisar as notificações de SMS e avisar o utilizador quando deteta padrões suspeitos. A análise é feita localmente no dispositivo.

3) Acesso à Internet

O acesso à Internet é usado apenas para descarregar regras de deteção (rulesets) que são assinadas criptograficamente. Estas atualizações não incluem dados do utilizador nem conteúdo de mensagens.

4) Armazenamento local

O SMS Seguro pode guardar localmente (no dispositivo):

  • Histórico de deteções (eventos e razões).
  • Domínios marcados como seguros (se o utilizador o fizer).
  • Rulesets de deteção (incluindo a versão atual).

Este conteúdo nunca sai do dispositivo e pode ser removido ao desinstalar a app.

5) Segurança

As atualizações de regras são verificadas através de assinatura digital (ex.: Ed25519) para reduzir risco de adulteração.
